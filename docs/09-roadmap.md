# 09 路线图

## 已完成（v0.1）

- 顾客菜单 + 购物车 + 提交订单
- 后台登录、订单列表、订单详情、状态流转
- 分类与菜品 CRUD
- 微信公众号测试号模板消息通知
- 通知日志表 + 后台查看
- **通知失败手动重发**
- **桌台二维码批量生成 + 打印**
- SSE 实时推送新订单 + Web Audio 提示音
- Docker Compose 一键部署

## 即将做（v0.2 候选）

- [ ] 菜品图片本地上传（multer + 静态目录），目前只接受外链
- [ ] 后台「今日营收」概览页
- [ ] i18n 框架接入（先繁化 / 英文化文案）

## 远期

- 多通知渠道（企业微信群机器人、Server 酱、PushPlus、邮件）
- 厨房显示页 `/kitchen`
- 微信小程序版菜单
- 在线支付（微信支付 V3）
- 多门店、多管理员账号、操作审计日志
- AI 菜单：根据人数推荐套餐 / 自动生成图片描述

## 已知限制

- 公众号测试号每日发送量受限，不适合正式商用
- 实时推送基于内存中的 `Set<Client>`，多实例部署需要换成 Redis Pub/Sub
- `OrderItem.unitPrice` 是下单快照；后台批量改价不会回写历史订单（这是预期行为）
